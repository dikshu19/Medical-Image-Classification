# Medical-Image-Classification
This project classifies medical MRI images using Deep Learning and Transfer Learning.

## Technologies Used
- Python
- TensorFlow
- MobileNetV2
- OpenCV
- NumPy
- Matplotlib

## Features
- Image preprocessing
- Data augmentation
- Transfer learning
- Model evaluation
- Prediction system

## Dataset
Brain Tumor MRI Dataset from Kaggle

## Project Output

- Model successfully classifies MRI images
- Prediction generated using Deep Learning
- Transfer Learning with MobileNetV2 used
